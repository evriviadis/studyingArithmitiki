# Practice Exercise 6: System of Nonlinear Equations

**Subject:** Solving multiple non-linear equations simultaneously (`fsolve`).

Δίνεται το παρακάτω σύστημα μη γραμμικών εξισώσεων:

1. Η εξίσωση ενός κύκλου: $x^2 + y^2 = 4$
2. Η εξίσωση μιας υπερβολής: $x \cdot y = 1$

**Ζητούμενα:**
1. Γράψτε ένα script στο MATLAB που να επιλύει το σύστημα αυτό με τη χρήση της συνάρτησης `fsolve`.
2. Φέρτε το σύστημα στη μορφή $F(v) = 0$, όπου $v = [x, y]^T$.
3. Αναζητήστε τη ρίζα που βρίσκεται στο πρώτο τεταρτημόριο, δίνοντας ως αρχική πρόβλεψη το διάνυσμα $x_0 = [1.5, 1.0]$.
4. Τυπώστε τα αποτελέσματα ($x$ και $y$) στο Command Window.
